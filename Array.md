Array
=====

From ES6:
---------

+ Array.from()
+ Array.of()
+ Array.prototype.repeat()
+ Array.prototype.sorted()

On es6pi:
---------

+ Array.prototype.repeat()
+ Array.prototype.sorted()

And `Array.`*fun* for corresponding `Array.`prototype.*fun* like Firefox
like `Array.sort`
