Set
===

From ES6:
---------

+ `Set`
 + Set.prototype.add()
 + Set.prototype.has()
 + Set.prototype.add()
 + Set.prototype.delete()
 + Set.prototype.values()
 + Set.prototype.clear()

When polyfilled by es2pi, Set is implemented as a child prototype of `Map`
which happens to have access to its keys only.
